Multi-Robot Logo Drawing in Turtlesim

This project illustrates a comprehensive approach to multi-robot control in a simulated environment, using PID controllers for navigation, ROS topics for communication, and services for environment management. The design is modular, allowing scalability in terms of the number of robots, and offers flexibility in defining navigation paths through goal points.

Lidar Based People Detection and Tracking System

The designed system utilized LIDAR data to detect and track people moving in a scene. It was built upon the ROS2 framework, leveraging the power of nodes, topics, and custom messages to process raw sensor data and output meaningful information about the environment.

Differential drive robot simulator in ROS

This project aimed to develop a differential drive robot simulator with ROS2, focusing on robot movement while accounting for wheel distance, velocity variance, and update rate errors.

LiDAR and Occupancy Grid Integrator in ROS

This project enhanced a differential drive robot simulator by integrating an occupancy grid map and a LiDAR sensor, significantly improving the simulation's realism and functionality.

Autonomous Navigation of Differential Drive Robots Using Reactive Control

This project develops a ROS-based simulator for a differential drive robot, focusing on autonomous navigation within various environments using reactive control strategies. Incorporating an occupancy grid map and a simulated LiDAR sensor, the system enhances simulation fidelity, enabling the robot to make informed navigation decisions, detect obstacles, and adjust its trajectory in real-time.

AUtonomous Navigation and Planning using PRM

This project develops an autonomous navigation system for a differential drive robot using the ROS framework. It integrates advanced path planning with real-time obstacle detection and avoidance, leveraging the Probabilistic Roadmap (PRM) method and simulated LiDAR sensor data.

Pathfinding Algorithms: Dijkstra vs A* on TAMU Rellis Center Map

This project presents a detailed comparison of two fundamental pathfinding algorithms, Dijkstra and A*, using a custom-created occupancy grid based on the layout of the TAMU Rellis Center. It aims to investigate the efficiency and effectiveness of each algorithm in navigating complex environments.

Planning and Control of an Autonomous Driving Vehicle in CARLA Simulator

This project features global route planning using the A* algorithm for efficient waypoint navigation, autonomous driving with PID-based speed control and Pure Pursuit for steering accuracy, and comprehensive simulation tests in CARLA simulator.

3d Printing using Articulated Robots

This project entailed enhancing a Yaskawa GP50 industrial robot's capabilities for construction-related tasks through advanced real-time control and motion planning. Utilizing ROS and Python, I developed a custom MoveIt package for precise trajectory execution, formulated ROS nodes to calculate and manage joint dynamics and end effector positions, and crafted controllers for seamless robot interaction.

Image Classification using ESRGAN and Deep Learning

This project investigates the impact of image resolution on classification accuracy through a two-phase deep learning process. Utilizing the CIFAR-10 dataset, images were first enhanced to high-resolution using ESRGAN, followed by classification into 10 categories via a Convolutional Neural Network (CNN).

Steel Sample Detection Using Transfer learning Approach - VGG16

This project leverages transfer learning from the VGG16 Convolutional Neural Network for feature extraction in steel micrograph images, coupled with SVM for binary classification.
